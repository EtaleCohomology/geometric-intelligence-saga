# Tools

GI (Geometric Intelligence) 理論の応用に関連する、データ分析・可視化のための実践的な研修教材群。

経済安全保障・地政学リスク・サイバーセキュリティの実務で活用できる、無料の公開データを使った可視化・分析手法を、段階的に学べる体系的な教材を提供する。

## 学習ロードマップ

データ分析の経験が浅い実務家から、GI 理論を活用したいデータサイエンティストまで、4 段階で着実にステップアップできる設計になっている。

### Step 1: ユークリッド空間でのビジネス可視化 (Tableau)

まず、平らな (ユークリッド) 空間における基本的なデータ可視化から始める。Excel ユーザーが Tableau Public のドラッグ&ドロップ操作で、経済安全保障データのダッシュボードを構築する。

- 対象: Excel ユーザー、データ分析未経験者
- 教材: [01_tableau-economic-security/](01_tableau-economic-security/)
- 学ぶこと: 棒グラフ、地図、Box Plot、時系列、統合ダッシュボード
- 数学的前提: 不要 (中学数学レベル)

### Step 2: ユークリッド空間でのデータ分析の自動化 (Python)

Tableau のドラッグ&ドロップ操作では限界がある分析 (大量データ処理、複雑な前処理、機械学習との連携) を、Python による自動化に拡張する。

- 対象: Tableau の基礎を理解したエンジニア
- 教材: 02_python-data-analysis/ (公開準備中)
- 学ぶこと: pandas でのデータ操作、seaborn での可視化、networkx でのネットワーク分析
- 数学的前提: 微分積分、行列計算 (大学初年級レベル)

### Step 3: 曲がったリーマン多様体上の計算 (Python 多様体プログラミング)

Step 2 までは「平らな空間 (ユークリッド空間)」での分析だった。しかし現実の経営環境やリスク環境は「曲がった空間 (リーマン多様体)」として捉えた方が、リスクの構造を正確に把握できる。Step 3 では、平らでない空間上での計算 (距離、曲率、測地線、ベクトル場のリー微分) を Python で実装する技術を学ぶ。

- 対象: Step 2 を終えた機械学習エンジニア・データサイエンティスト
- 教材: (公開準備中)
- 学ぶこと: 変分オートエンコーダ (VAE) による多様体構築、プルバック計量、スカラー曲率、リー微分
- 数学的前提: 線形代数、確率論、ニューラルネットワークの基礎

### Step 4: 経営環境多様体上の AI エージェント・最適制御 (GI 理論)

GI 理論 (Geometric Intelligence) の本格応用編。曲がった多様体上で動く AI エージェント、最適制御、形式検証、ZKML を組み合わせ、企業や政策立案者向けの意思決定支援システムを構築する。

- 対象: Step 3 を終えた研究者・実務家
- 教材: [theory/](../theory/) (理論編 Vol.1, Vol.2)、[case-studies/](../case-studies/) (応用ケース)
- 学ぶこと: ポントリャーギンの最大値原理、シンプレクティック積分、層理論、手術理論、形式検証 (Coq/Z3)
- 数学的前提: Step 3 の内容に加え、最適化理論の基礎

## なぜこの順番か

Step 1 と Step 2 は、いわば**地図の読み方を学ぶ段階**。平らな世界での観測手法を体得する。

Step 3 と Step 4 は、**地図そのものを描き直す段階**。世界が実は曲がっていることを認め、その曲がった世界を航行する技術を学ぶ。

GI 理論を Step 4 から学ぼうとすると、多くの実務家は数学的負荷に挫折する。Step 1 → 2 → 3 → 4 と段階的に学ぶことで、実務家でも GI 理論の応用ができるレベルに到達できるよう設計されている。

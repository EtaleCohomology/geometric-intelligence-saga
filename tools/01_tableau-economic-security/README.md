# Step 1: Tableau による経済安全保障データ分析

**Excel 卒業生のためのデータ分析入門 — 全 12 回研修教材**

GI (Geometric Intelligence) 理論への学習ロードマップにおける **Step 1**。曲がった多様体に進む前に、まず平らな (ユークリッド) 空間での基本的な可視化技術を体得する段階の教材。

## 概要

経済安全保障・地政学リスク・サイバーセキュリティの実務で扱うデータ ── OFAC 制裁リスト、米国 Entity List、OECD カントリーリスク、財務省貿易統計、CISA KEV など ── を、Tableau Public のドラッグ&ドロップ操作だけで可視化・分析できるようにする実践的な研修教材。

すべて無料で利用できる公開データソースを使い、Tableau Public (無料版) のみで完結する設計になっている。

## 対象読者

- Excel は使えるが Tableau は触ったことがない実務家
- 経済安全保障・サプライチェーンリスク・サイバー脅威に業務で触れる調達・営業・コンプライアンス担当者
- 「制裁リストを毎月チェック」が今は手作業の方
- Python は書けないが、ドラッグ&ドロップでなら何とかなりそうな方

## 学習目標

本教材を完了すると、以下ができるようになる:

- 12,000 件超の OFAC SDN リストを世界地図にプロットできる
- Box Plot でカントリーリスクの分布を読み解ける
- 時系列グラフに地政学イベントを参照線として重ねられる
- 4 つのワークシートを連動させた統合ダッシュボードを構築できる
- 制裁対象企業の子会社・フロント企業をネットワーク図で可視化できる
- CISA KEV のサイバーセキュリティ脆弱性データを可視化できる

## 構成 (全 12 回)

### 基礎編 (第 1〜6 回)
- 第 1 回: Tableau Public のセットアップと基本操作
- 第 2 回: OFAC SDN リストを世界地図にプロット
- 第 3 回: 制裁プログラム別フィルタリング
- 第 4 回: OECD カントリーリスクを Box Plot で可視化
- 第 5 回: 財務省貿易統計の時系列分析
- 第 6 回: 地政学イベント参照線とトレンドライン

### 応用編 (第 7〜11 回)
- 第 7 回: ネットワーク分析 — 制裁対象企業の子会社・フロント企業を可視化
- 第 8 回: 時系列予測 — Tableau 標準機能で 12 か月先のリスク予測
- 第 9 回: 多変量解析 — TabPy 連携で PCA・クラスタリング

## 使用するデータソース

すべて無料で利用できる公開データのみ。本教材は **教育目的** で各データソースを参照する方法を示すもので、データ自体を再配布するものではない。各データソースは、演習の指示に従って公式サイトから直接ダウンロードする。

| データソース | 提供元 | ライセンス |
| --- | --- | --- |
| OFAC SDN リスト | 米国財務省 | 米国政府著作物 (自由に利用可能) |
| Entity List | 米国商務省 | 米国政府著作物 (自由に利用可能) |
| CISA KEV | 米国 CISA | 米国政府著作物 (自由に利用可能) |
| NVD CVE | 米国 NIST | 米国政府著作物 (自由に利用可能) |
| World Bank WGI | 世界銀行 | CC BY 4.0 |
| OECD Country Risk | OECD | 教育目的での参考使用が可能 (再配布不可) |
| 財務省貿易統計 | 日本財務省 | 公開統計として一般利用可能 |

業務で実際に活用される際は、各データソースの最新の利用規約を必ずご確認ください。商用ツール・サービスへの組み込みには、別途ライセンス契約が必要な場合があります。

## 必要なツール

- **Tableau Public** (無料): https://public.tableau.com からダウンロード
- インターネット接続 (公開データのダウンロード用)

教材本体は Tableau Desktop (有償版) ではなく、Tableau Public (無料版) で完結する設計になっている。

## 注意事項

Tableau Public は無料だが、作成したダッシュボードはインターネット上に公開される。社内データを扱う場合は Tableau Desktop (有償) または Tableau Cloud を使用すること。本教材の演習はすべて公開データを使うので、Tableau Public で問題ない。

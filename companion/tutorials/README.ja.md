# チュートリアル

ジオメトリック・インテリジェンス(GI)理論を新規読者に紹介する教育資料。

## 目的

本ディレクトリのチュートリアルは、データサイエンス一般に親しみのある読者と、GI理論Volume 1・2で用いられる数学的機構との橋渡しをする入門資料である。高校数学を前提とし、必要な微分幾何学の語彙(リーマン計量、Lie微分、測地線、最適制御)を、比喩、具体例、Python実装の断片を通じて構築する。

本チュートリアルはVolume 1・2の代替ではない。読者が本理論に取り組むための準備を整え、また概念をすばやく参照するためのリファレンスとしても機能する。

## 収録内容

自己完結型の入門資料。以下を扱う:

- 第3章: 物理・化学・工学がこの数学を100年間使ってきた事実
- 第4章: なぜ社会経済データではこのアプローチが今まで利用できなかったか

- 第6章: PyTorch、torchdiffeq、pydiffmap、POT等を用いたPython実装例

日本語のみ(現時点)。約48ページ。

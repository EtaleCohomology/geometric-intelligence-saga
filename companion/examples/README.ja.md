# ケーススタディ

ジオメトリック・インテリジェンス(GI)理論のコンパニオン巻で示した10ステップ・パイプラインの完全実装事例。

## 目的

各ケーススタディは、データ取り込みから最終的な経営判断に至るまで、パイプライン全体を架空だが妥当性のある企業・政策シナリオに適用する。データは合成データであり、既知の潜在構造から生成されているため、幾何学的分析をground truthに対して検証できる。各事例は、GI理論の数学的機構が具体的な経営上の洞察にどのように翻訳されるかを示すよう設計されている。

## スコープ

これらは玩具のような問題ではない。各ケーススタディは現実的な指標セット(通常40〜50変数、6〜10カテゴリ)、10年間の月次時系列、コンパニオン巻で規定される全品質チェックを含む。目標は再現性にある:パイプラインに従う読者は、報告された結果と整合する結果を得られるはずである。

## 現在の事例

### `case_study_01_yamato_foods_*`

**大和フーズ株式会社——レトルト食品メーカー、6市場展開の意思決定**

インド市場参入を検討する架空の東京本社レトルト食品メーカー。50指標×8カテゴリ、6市場、10年分の月次データ(720観測点)。パイプラインは以下を生成する:潜在多様体の構築、引き戻し計量、日本/米国クラスターとインド/ブラジルクラスターの間に構造遷移帯を同定するスカラー曲率マップ、3つの政策候補のLie微分分析、30ヶ月の最適制御スケジュール。取締役会決議は、幾何学的に最適な経路を採用し、リスクバッファを明示的に計上する。

ファイル:
- `case_study_01_yamato_foods_{JA,EN}.pdf` — パイプライン実行報告書
- `case_study_01_yamato_foods_data_{JA,EN}.pdf` — データセット詳細(指標、架空SNS投稿、架空メディア記事見出し、架空規制文書)

## 合成データについて

本ケーススタディの全データは合成データである。企業名、SNS投稿、メディア見出し、規制文書は、説明目的で創作されたものである。実在機関名(日経、Bloomberg、FDA、FSSAI等)は出典ラベルとして用いているが、文書自体は架空である。これは各ケーススタディ内で明示されている。

合成データは、幾何学的分析を既知のground truthに対して検証できるという利点があり、教材として合成データを用いる主たる理由である。

# 企業編 ケーススタディ

業界別の企業ケーススタディ。各業界における経営危機への GI (Geometric Intelligence) 理論の応用を物語形式で示す。

本シリーズは、GI 理論を実際の企業経営環境データに適用する際にデータサイエンティストが直面する具体的な課題と、その解決のための創意工夫を、ケーススタディ・ノベルの形式で解説する実践的な教材である。

### [textile-industry/](textile-industry/) — 繊維業界編
**生存の幾何学 — 綾鳥テキスタイル株式会社 14ヶ月の戦争**

架空の中堅メーカーが、国家支援型 APT グループのランサムウェア、新疆綿エンティティリスト追加、AI 生成偽造品、EU デジタル・プロダクト・パスポート規制、敵対的買収、内部情報漏洩、台風、新型呼吸器ウイルスに同時多発的に直面する 14 ヶ月の物語。

#### 本作で扱うデータ解析の現場課題

GI 理論の理論的フレームワークと、実際の経営環境データへの適用との間には、データサイエンティストが現場で直面する数多くの課題が存在する。本作はこれらを物語の中で具体的に解説する。

**データ収集と前処理の課題**
- 8 カテゴリ・約 100 指標・8 年分のデータを、複数の異なるシステム (SAP / Oracle / 自社開発) から統合する困難
- 412 セル (0.74%) の欠損データへの対応 ── 「どの欠損は補完すべきで、どの欠損は補完してはいけないか」の判断
- COVID-19、スエズ運河封鎖、ウクライナ戦争など「レジームシフト」期間の扱い ── 通常の補完手法を適用すると本物の不連続性を滑らかに均してしまう問題
- 工場の現地経営陣交代により紙の請求書としてしか存在しないデータを 3 日間でデジタル化する現場対応
- 同一指標でも拠点によって測定方法が異なる (リードタイムの定義の違いなど)

**多様体構築における数学的危機の解決**
- 命題 1.1 (再構成忠実度・潜在空間の滑らかさ・ヤコビアンの非退化性) の検証で 3 条件中 2 条件が不合格となった際の原因究明
- 潜在空間の t-SNE 射影により、データが 3 つの孤立クラスター (COVID 前・撹乱期・後) を形成していることを発見する診断プロセス
- 物理情報に基づく Neural ODE による制約付き補間で「合成境界データ」を生成する手法 ── 捏造データとの本質的な違いの説明
- ホイットニーの拡張定理による合成データ生成の数学的正当化

**実務の苦労と業務調整**
- クライアント企業の CFO が「高等数学は健全な財務判断の代わりにはならない」と懐疑的な中で、データ品質の重要性 (ショートブレッドの教訓) を説明する場面
- 月次データしかないシステムから日次粒度のトランザクションレコード 287,000 件を 24 時間で抽出してもらう交渉
- 機密データ (政府保有のサイバー脅威インテリジェンス) と非機密データの統合における準同型埋め込みの実装

**緊急時のデータサイエンティストの判断**
- ランサムウェア攻撃による工場停止という現実世界の脅威の最中に、48 時間でリー微分計算を完了させる現場対応
- 多変量解析の結果が直感に反する処方 (生産復旧より先にブランド防衛) を示した際に、CFO に統計的根拠を説明する困難
- 全予測に MC ドロップアウトによる信頼区間を付与し、「知っていることと知らないことを分ける」原則の実践

**敵対的環境下でのモデル運用**
- 内部情報漏洩により敵対者が我々の多様体分析を入手した可能性が判明した際のゲージ変換による対応
- ソフィアの大学アカウント経由のスピアフィッシング攻撃から、メリディアン社自身のシステムを守る SDN セグメンテーション
- 48 時間ごとの座標系ローテーションの運用設計

**長期戦略の数学的根拠の提示**
- 取締役会への 5 年予測において、シンプレクティック積分器 (シュテルマー＝ヴェルレ法) を採用する理由 ── 通常の数値積分器ではエネルギードリフトが累積する問題
- 形式検証 (Coq / Z3 / Agda) による「単月で企業価値を 5% 超低下させる推奨は出せない」という安全性の数学的証明
- ZKML (ゼロ知識機械学習) による「モデルや機密データを開示せずに計算の正しさを証明する」仕組みの実装

#### 想定読者

- 製造業・商社・金融機関・コンサルティングファームのデータサイエンティスト
- 機械学習エンジニア (特に PyTorch・scikit-learn・networkx の実務者)
- 経営企画・リスク管理・コンプライアンス担当者
- 数学的アプローチに関心のある CFO・CRO
- 微分幾何学・代数的位相幾何学を学んだ研究者で、産業応用に関心のある方

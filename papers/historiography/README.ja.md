# 歴史学方法論

**「歴史家の多様体——E.H. カーとリーマン幾何学の対話」**

## 概要

本サブディレクトリは、単一の作品の4言語版を収録する。すなわち、E.H. カー、アナール学派、ベネディクト・アンダーソンの歴史学方法論的伝統と、ジオメトリック・インテリジェンス(GI)理論のリーマン幾何学的装置との間の哲学的対話である。本作は、歴史的多様体の解釈依存性についての方法論的反省を構成する——すなわち、同一のデータは、目的関数の変更のもとでは、幾何学的に異なる多様体を生成するという命題であり、カーの言う歴史家の視座が、GI理論における目的関数と項ごとに対応するという命題である。

対話は、著者のGIケーススタディ・ノベル群の登場人物たち——リルル先生、倉田教授、カイト、遠藤真紀、リン、リーム——によって展開される。舞台は2036年秋、ケンブリッジ大学トリニティ・コレッジのリルル先生の研究室。議論は4つの定式化を順次辿る。第一に、カーの言う歴史家の問いが、GI理論の目的関数と対応すること。第二に、この対応が多様体上のモース関数の選択にまで拡張され、歴史が二重に解釈依存的であることが示される。第三に、複数の目的関数の並行実行から生じる多様体の多重性が、ゲーデルの不完全性定理への実践的応答として提示される。第四に、アナール学派の集合心性(mentalité collective)、ベネディクト・アンダーソンの想像の共同体、そしてGI理論の多様体が、単一の現象の3つの記述言語として構造的に同一であることが明らかにされる。

本作には新規の数値計算結果は含まれない。本作は全編を通じて、既存のGI理論の数学的装置——目的関数、多様体、スカラー曲率、Lie微分、モース関数——を、20世紀歴史学方法論の光のもとで概念的に再解釈する試みである。

## GIコーパス内での位置づけ

本作は、GIケーススタディ・ノベル・シリーズの最初の番外編(bangai-hen)である。2036年秋——ノベル『リームの過ち』の出来事から半年後、Eight Eyes多様体の拡張が完了し、新しい測地線が承認された後の、静かな知的再建の時間を舞台とする。『リームの過ち』の既読は前提としない。哲学的議論は自立しているが、リームのコンゴ体験——アマニ、カヤンバ——への言及は、先行ノベルを読むことで最大の共鳴を獲得する。

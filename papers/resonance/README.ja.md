# 構造的共鳴

**ジオメトリック・インテリジェンス(GI)理論と20世紀思想の諸伝統との構造的共鳴について**

## カテゴリについて

本サブディレクトリは、著者が **構造的共鳴(Strukturresonanz)** と呼ぶものを扱う論考を収録する。すなわち、翻訳でも形式化でも還元でもない、ある種の関係——GI理論が記述する数学的現象と、人間の経験の時間的・空間的構造を問うてきた思想の諸伝統が、それぞれ異なる言語で記述する現象との間に成立する関係——である。本ディレクトリの諸論考は、ベルクソンの持続を、ハイデガーの世界内存在を、木村敏の「あいだ」を、コルバンの感性の歴史を、可微分多様体とLie微分の語彙に翻訳しようとはしない。それらが試みるのは、より控えめでありながら同時により要求水準の高い営みである——すなわち、GI理論の数学的装置と、これらの伝統の概念装置とが、根本的に異質であるにもかかわらず、同一の構造的現象を指し示す点に、耳を傾けること。各伝統はそれぞれの語法で、それぞれの視座から、自らの尊厳を失うことなく、同じ現象を指し示す。

したがって本カテゴリの方法論的立場は、翻訳者のそれではなく**聴き手**のそれである。著者は、Lie微分が純粋持続の本質を捉えると主張しない。リーマン多様体上の測地線距離が木村の「あいだ」を再構成すると主張しない。そのような主張は、還元主義的傲慢の形態として正しく退けられるべきものである。ここで肯定されるのは、これらの分野の比較不可能な表層の下に、地下水のような伝達が流れているということ、そして本論考の任務は、その伝達が聴き取られうるところならばどこでも、耳を澄ますことである。

本カテゴリは開かれた系列として構想されている。ガダマーの地平の融合と歴史的多様体、カッシーラーの象徴形式と潜在空間の次元性、シモンドンの個体化とLie括弧の動力学、その他の類似の共鳴——これらの論考が、プログラムの展開とともに追加されていく予定である。

## 収録内容

### `01_duration_and_inbetween/`

**「持続と『あいだ』の幾何学——ベルクソン・ハイデガー=ツォリコーン・木村敏=京都学派・コルバン=アナール学派——そしてGI理論」**

本カテゴリの最初の論考。ベルクソンの純粋持続、ハイデガーの世界内存在とツォリコーン・ゼミナール、木村敏の「あいだ」と京都学派、コルバンの感性の歴史——これら4つの知的伝統と、GI理論の可微分多様体、Lie微分、スカラー曲率、モース関数、測地線距離——との間の構造的共鳴を辿る試論。西田幾多郎の「絶対矛盾的自己同一」によって枠取られ、ラングランズ・プログラム、丸山善宏の圏論的統一科学、望月新一の宇宙際タイヒミュラー理論を、数学的知の統一をめぐる21世紀の開かれた問いとして考察する末尾で結ばれる。

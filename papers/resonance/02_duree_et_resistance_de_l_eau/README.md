# 02. 水流の抵抗と二つの多様体 —— ベルクソンとリーマンの間で

> **構想メモ段階** — 本論考は現在、構想メモの段階にある。本文書は、将来の本論文執筆に向けた予備的考察を記録したものである。

## 概要

本稿は、`papers/resonance/` シリーズの第二章として構想されている論考の予備的スケッチである。

ベルクソンの「持続(durée)」概念とリーマンの「多様体(Mannigfaltigkeit)」概念を、二十一世紀のフィンスラー幾何学・最適輸送理論を経由して再訪する試みのための、最初の種である。

具体的事例として「水流の抵抗」を取り上げる。水の中で泳ぐ身体の経験は、流体力学の方程式で完全に記述できるか。それとも、数値化を逃れる質的な何かが残るか。この問いを通じて、二人の思想家の間にあるかもしれない橋を探る。

## 中心的な問い

本稿が育てようとしている、三つの種となる問い:

1. 水の抵抗を、リーマン計量で記述すれば失われるものは何か?
2. β 項(風や水流)が時間とともに変化することと、ベルクソンの「持続」は同じことか?
3. 泳ぎながら「次のストローク」を選ぶとき、それは最適制御か、それとも持続の中での自由な選択か?

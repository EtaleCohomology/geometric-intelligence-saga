# 持続と「あいだ」の幾何学

**ベルクソン・ハイデガー=ツォリコーン・木村敏=京都学派・コルバン=アナール学派——そしてGI理論**

## 概要

本サブディレクトリは、単一の試論の6言語版を収録する。すなわち、人間の経験の時間的・空間的構造を問うてきた4つの知的伝統——ベルクソンの純粋持続、メダルト・ボスとのツォリコーン・ゼミナールにおいて展開されたハイデガーの世界内存在、京都学派の系譜において練り上げられた木村敏の「あいだ」概念、アナール学派の伝統におけるアラン・コルバンの感性の歴史——と、他方における、ジオメトリック・インテリジェンス理論の数学的装置——可微分多様体、Lie微分、スカラー曲率、モース関数、測地線距離——との間の構造的共鳴を辿る論考である。

本論考が採る方法は、翻訳でも形式化でも還元でもなく、著者が **構造的共鳴(Strukturresonanz)** と呼ぶものである。すなわち、数学的なものと哲学的なものとが、その語法の根本的な非通約性にもかかわらず、同一の構造的現象を指し示す点を、受け取ること。Lie微分は純粋持続の本質を捉えない。測地線距離は木村の「あいだ」を再構成しない。そのような主張は一切なされない。主張されるのはただ、これらの非通約的表層の下に、地下水のような伝達が流れているということ、そしてこの試論の任務は、その伝達が聴き取られうるところならばどこでも、耳を澄ますことである。

議論は、西田幾多郎の「絶対矛盾的自己同一」——互いに矛盾する二つのものが、その矛盾の只中で一であるという命題——によって枠取られる。これは、本試論が提示する数学と哲学の関係についての最終的な形姿である。ハイデガーの言葉を借りれば、**存在の二つの家が、同じ大地の上に立っている**。

末尾の数ページは、考察をラングランズ・プログラム、丸山善宏の圏論的統一科学、望月新一の宇宙際タイヒミュラー理論へと拡張し、これらを**知の新たな統一の可能性**をめぐる21世紀数学の開かれた問いとして位置づける。中世の自由七科の地図を引き裂いた近代の学問的分業を、これらのプログラムの進展が示唆しはじめた糸によって、再び縫合することはできるのか——という問いとして。

## 本作の性格について

本試論は、著者のこれまでの著作の中で**最も個人的なもの**である。その議論は全編を通じて哲学的・数学的なものであるが、序文には、著者が本論の主題を初めて出会うに至った伝記的経験が記録されている——意味を把握しえぬうちから暗唱させられた幼稚園期のサン=テグジュペリ『星の王子さま』フランス語原文、少年期にポール・フルキエ『哲学精要』に出会ったこと、ミンコフスキー『生きられる時間』とメルロ=ポンティ『知覚の現象学』を読んだこと、ニコラ・ブルバキの物語との邂逅、そしてアイルランド人、イギリス人、アメリカ人、フランス人、ロシア人、トルコ人、正統派ユダヤ人の友人たちのあいだで育った体験——本試論が立てる構造的問いは、哲学的形姿を取るよりもはるか以前に、まず身体のレベルで出会われたものであった。

この意味で、本作は、表題に掲げられた主題についての試論であると同時に、**著者の知的自伝**である——すなわち、一人の人間が、数学と人文学とがその同一の精神の生において互いに異邦人でなくなりうると信ずるに至った、地下の道筋の記録である。

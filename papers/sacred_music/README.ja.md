# 宗教音楽

**「ジオメトリック・インテリジェンス理論の宗教音楽への適用」**

## 概要

本ディレクトリは、ジオメトリック・インテリジェンス(GI)理論のリーマン幾何学的枠組みを宗教音楽作品に適用したケーススタディを収容する。音楽——とりわけ西方キリスト教伝統の典礼音楽——は、利用可能な「感性的空間」(espace sensible)のなかでも最も純粋な事例の一つを提供する。人間の感情の最も繊細な構造が、幾世紀にもわたり、他の領域ではまず得がたい精度で刻み込まれ、伝承されてきた領域だからである。

本ディレクトリの方法論的射程は二つある。第一に、経済・政治・歴史の構造分析のために開発された数学的装置——変分オートエンコーダ(VAE)のデコーダの引き戻し計量、Lie微分、スカラー曲率、測地線方程式——が、音楽の楽譜に対しても意味ある適用を許すことを示す。第二に、その適用が、音楽を数値的骨格へと還元するのではなく、むしろ、演奏者と聴取者が通例は暗黙裡にのみ気づいていてその言語化に至らなかった構造を——知覚的表層の下に潜む構造を——分析的に可視化することを提案する。

## サブディレクトリ

### `ave_verum_corpus/`

モーツァルトのモテット『Ave Verum Corpus』(K.618、1791年6月)——作曲者自身の死の6ヶ月前、バーデン=バイ=ウィーンの小さな教区における聖体の祝日のために作曲された46小節。本サブディレクトリは2つの研究を収録する。第一の研究は、半分架空のデータセットに基づく方法論的提案である。第二の研究は、公開された楽譜(IMSLP)からのMIDI転写データに基づく実証的検証である。日本語版、英語版、ドイツ語版、フランス語版、ラテン語版を収録。
